### 2.2 Ключевые акценты для удержания внимания

1. **Live-demo на 4-й минуте** — открываем браузер, члены жюри подключаются с телефонов по QR / коду комнаты, играем мини-партию.
2. **Цифры на слайдах**, а не на слайдах + в речи (не дублировать).
3. **Истории CustDev** — два-три коротких живых анекдота из интервью (Маша обиделась, что её каждый раз делают ведущей; администратор антикафе сказал «забираю как замену зарплате»).
4. **Никаких «команда мечты»** — три роли, три имени, переход.

Поддерживает оба сценария (офлайн в одной комнате через коды и онлайн — игроки из разных мест). Телефон у каждого уже есть; React + WebSocket дают мгновенную синхронизацию фаз. ## Шаг 5. Обратная связь и фиксация улучшений

### 5.1 Три сильные стороны

1. **Live-demo на 4-й минуте** — единственная команда из питч-сессии, у кого продакшн-prototype работает прямо в зале и его можно потрогать.
2. **Прозрачная воронка цифр Market** — расчёт TAM/SAM/SOM читается без расшифровки, допущения подписаны.
3. **Конкретный Ask** — 1,2 млн ₽, 6 мес., 5 направлений, 2 KPI; не «дайте денег на развитие».

### 5.2 Три зоны улучшения

1. **Слайд Competition** — матрица понятна, но не показывает, насколько у конкурентов слабее именно «характер голосового ведущего». Добавить цитаты из CustDev на следующей итерации.
2. **B2B-канал** — на питче кратко, без расчётной модели. Подготовить отдельный one-pager для B2B-инвесторов / антикафе.
3. **Risk-слайд** отсутствует — добавить компактный список рисков и митигаций (5 пунктов на 1 слайд).

### 5.3 Три правки на следующей итерации

1. Добавить слайд **Risks & Mitigations** между Validation и Business model.
2. Заменить статичную картинку на слайде Solution на 6-секундный GIF демо-партии (на случай отказа live-demo).
3. Подготовить отдельный B2B one-pager (1 страница А4) с экономикой по антикафе.

## Шаг 7. Защита проекта

**Тезисы для защиты:**

- **Что показываем:** работающее веб-приложение (React + FastAPI + PostgreSQL + WebSocket), которое ведёт мафию голосом без живого ведущего; пакет коммерческих метрик пилота (5 партий, 24 сессии, 8 заявок от антикафе).
